# Görseller

Hero ve galeri şu an Unsplash stok görselleri kullanıyor.

Kendi frigolu araç / taşıma fotoğraflarınızı eklemek için:

1. JPG veya WebP dosyalarınızı bu klasöre koyun (ör. `frigo-van-1.jpg`).
2. `src/components/Hero.tsx` ve `src/components/Gallery.tsx` içindeki `src` URL’lerini `/images/frigo-van-1.jpg` gibi yollarla değiştirin.
